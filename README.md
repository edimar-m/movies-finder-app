 # Movies-Finder-app 🍿

  ### Esta aplicación te permite buscar peliculas y conocer acerca de ellas, te muestra las últimas películas lanzadas y las mejores rankeadas, a demás puedes ver sus trailers📽️🎞️.
  ### 

  ### Esta aplicación fue realizada con ReactJS la API The Movie Database, como evaluación final curso de desarrollo FrontEnd en ADA ITW.

  ### Si quieres buscar información de tus peliculas favoritas, puedes ingresar a la app [aquí]( https://edimar-m.github.io/movies-finder-app/).
